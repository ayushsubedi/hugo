Inclusive Education for People with Disability - Are We Doing Enough?

An analysis of the existing gaps in the government policies for inclusive education for people with disabilities.

Goal of the Project

** this is a sample. We want to examine if current government policies appropriately designed to facilitate school enrollment for people with disabilities. The goal of the project is to To successfully initiate an advocacy campaign designed to increase awareness and eventually lead to improvements in existing government policies.


Summary of the Project

** this is a sample. We realized that there is a need to push for investing in collecting and disseminating data for this context. Also, there is a need to start discussions with different line ministries to integrate disability indicators in the IS. Similarly, gaps in government policies for inclusiveness should be studied.
